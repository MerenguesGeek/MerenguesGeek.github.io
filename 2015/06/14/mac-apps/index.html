Mac下的神器

By Zec

发表于 2015-06-14

文章目录

开发篇

  • Alfred：说它是 Mac 下第一神器估计没什么人反对吧，我觉得任何解释对于它都是多余的，谁用谁知道。（£ 17 / License）
  • Dash：程序员查文档什么的最烦了，Dash正是一个把您从烦恼中解救出来的神器，没有您想不到的语言，只有您不会用的语言。（$ 19.99 / License）
  • Kaleidoscope：代码比对合并软件，个人认为是 Mac 上 UI 最漂亮的代码比对合并软件。（Mac App Store：￥ 448）
  • Spark Inspector：UI 调试软件，可以将 app 的所有图层用 2D / 3D 的样式显示出来，方便调试 UI。（$ 49.99 / License）
  • Mou：Mac 下的 markdown 编辑工具，支持实时浏览，可将文档导出为 HTML 和 PDF 格式。
  • GitHub：全球最大同性社交平台 / 全球最大的代码托管平台，只需要这两个头衔就够了。
  • Charles：网络调试工具，可以利用 Mac 共享 WiFi 网络给 iPhone / iPad，再通过 Charles 抓取 iPhone / iPad 的网络数据。
  • Jason：强大的 JSON 数据解析器，自动分析数据的类型，方便开发者进行网络数据解析。

生产力篇

  • 1Password：密码保护软件，可以多平台同步，如果您是苹果设备的忠实粉丝，那么iPhone和Mac设备件无缝的密码同步简直不能太爽。（Mac App Store：￥ 328）
  • Evernote：全平台共享文档和笔记，安装浏览器插件后可通过截屏等方式保存网页，目前最好的网络笔记本。
  • Caffeine：控制 Mac 休眠的小软件，一键设置，可以免除许多不必要的操作。
  • Reeder：虽然 Google Reader 关闭后大家一致认为 RSS 订阅已死，但是订阅一些精品的技术和行业博客依然是我最热衷的获取资讯的方式。Reeder 满足我对 RSS 订阅器的所有要求。（Mac App Store：￥ 68）
  • 奇妙清单：拖延症患者必备。
  • Sketch：移动 UI 设计师必备，尤其是一键导出 @2x、@3x 后缀的图片，太有爱了。（Mac App Store：￥ 648）
  • MindNode Pro：思维导图工具，相对功能较简单，但已能满足我的日常需要。（Mac App Store：￥ 128）
  • HyperDock：虽然 Mac 有了多桌面，但是界面之间的切换还是不方便，有了 HyperDock 配合多桌面就方便多了。（Mac App Store：￥ 68）
